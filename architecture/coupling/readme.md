# Cohesion, Coupling and Cost

A short presentation I wrote on architecture principles and their drivers.

## Build

* **VSCode**  
`CTRL+SHIFT+B` and select `Build coupling presentation`.

* **Powershell**  
Run `./build.ps1`.

## Presenting

* Open `onstage.html`.
* Open pop-up.
* Maximise pop-up with `F11`.
* Share pop-up.
