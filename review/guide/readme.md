# Code Review Guide

A short presentation I wrote for making code reviews a positive experience.

## Build

* **VSCode**  
`CTRL+SHIFT+B` and select `Build review guide presentation`.

* **Powershell**  
Run `./build.ps1`.

## Presenting

* Open `onstage.html`.
* Open pop-up.
* Maximise pop-up with `F11`.
* Share pop-up.
